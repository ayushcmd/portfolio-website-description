## ✨ Features

- **3D intro** — "AYUSH RAJ" assembles from blocks, holds, then explodes (Three.js)
- **SPA architecture** — section-based navigation, zero page reloads
- **Custom cursor** — dot + ring with hover expansion effect
- **Typewriter** — cycles through what I'm currently building
- **Skills tabs** — 4 panels (Foundations / Languages / AI·ML / Tools) with proficiency badges + tooltips showing when each skill was learned
- **GitHub Activity Feed** — live public events from GitHub API
- **Quick Enquiry** — floating form powered by Formspree, sends direct to inbox
- **Scroll progress bar** — top-edge progress indicator
- **Keyboard navigation** — press `1`–`5` to jump sections
- **Mobile responsive** — hamburger menu, full sidebar collapse
- **Custom 404 page** — themed to match the portfolio
