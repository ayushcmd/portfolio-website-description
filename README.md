## Design

**Theme: The Grid Cartographer**

- Fixed left sidebar navigation (SPA architecture)
- Custom cursor system with ambient glow
- Animated SVG coordinate grid background
- Typography: Syne + Instrument Serif + JetBrains Mono
- Color palette: Deep ink · Electric amber · Acid mint

---

## Features

- Single-page application (SPA) with section-based navigation
- Skill tooltips showing when and how each skill was learned
- Quick Enquiry form powered by Formspree (sends directly to inbox)
- Sidebar collapse/expand toggle
- Custom animated cursor
- Mobile responsive with hamburger menu
- Privacy Policy page
- Custom 404 error page
